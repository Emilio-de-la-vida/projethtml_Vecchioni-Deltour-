Projets étudiants

Les projets permettent aux étudiants de mettre en pratique leurs connaissances, de travailler en équipe et de développer leur créativité.

Exemples de réalisations

Site vitrine responsive

Technologies : HTML, CSS, JavaScript

Création d’un site moderne avec design responsive et interactions dynamiques.

Application de gestion d’étudiants

Technologies : SQL, PHP, HTML

Développement d’un outil permettant de gérer des données et des profils utilisateurs.

Mini-jeu éducatif

Technologies : Python, Pygame

Conception d’un jeu interactif pour sensibiliser à des thématiques écologiques.

Analyse de données

Technologies : Python, CSV

Étude de jeux de données pour produire des indicateurs, graphiques et interprétations.

Pourquoi les projets sont importants

Les projets permettent d’apprendre autrement. Ils aident les étudiants à mieux comprendre les cours, à résoudre des problèmes concrets, à gérer leur temps et à présenter leur travail de manière structurée.
