À propos du projet

Ce site a été réalisé dans le cadre d’un projet académique à l’EFREI. Il a pour objectif de présenter le département Informatique à travers ses formations, son équipe enseignante, sa vie étudiante, ses projets et ses liens avec le monde professionnel.

Objectif du site

L’objectif de ce projet est de concevoir un site vitrine moderne, structuré et responsive, développé en HTML, CSS et JavaScript, afin de valoriser le département Informatique de l’EFREI.

Le site propose une navigation claire entre les différentes pages, un contenu organisé par thématiques, des éléments visuels, des tableaux, une carte interactive, une FAQ, ainsi que plusieurs fonctionnalités dynamiques comme le mode sombre et le bouton de retour en haut.

Technologies utilisées

HTML

Structure des pages, organisation du contenu et navigation entre les sections.

CSS

Design, mise en page, responsive design, cartes, boutons et identité visuelle.

JavaScript

Ajout d’interactivité : mode sombre mémorisé, FAQ interactive, carrousel, filtres de projets et bouton retour en haut.

VS Code

Environnement de développement utilisé pour créer, organiser et tester le site.

GitHub

Hébergement du dépôt du projet, suivi des versions et partage du code source.
