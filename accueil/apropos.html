À propos du projet

Ce site a été réalisé dans le cadre du projet final de programmation Web TI402. Notre objectif était de créer un site vitrine clair, moderne et agréable à utiliser pour présenter le département Informatique de l’EFREI.

Objectif du site

Nous avons voulu créer un site simple à comprendre, qui permet de trouver rapidement les informations importantes sur les formations, la vie étudiante, les projets, les événements et les admissions.

Le site a été réalisé en HTML, CSS et JavaScript, sans framework. Nous avons aussi ajouté plusieurs interactions pour rendre la navigation plus dynamique.

Pages réalisées

Accueil

Page principale du site avec une présentation du département, un carrousel, des chiffres clés et des accès rapides.

Formations

Page qui présente les différents parcours, les matières étudiées et les débouchés possibles.

Équipe & Recherche

Page qui met en avant l’équipe enseignante et les grands domaines liés à la recherche informatique.

Vie étudiante

Page dédiée au campus, aux associations, aux clubs et à l’ambiance étudiante.

Événements

Page qui regroupe les journées portes ouvertes, conférences et ateliers.

Projets étudiants

Page qui présente quelques exemples de projets réalisés par les étudiants.

Professionnalisation

Page consacrée aux stages, à l’alternance, aux entreprises partenaires et aux métiers possibles après la formation.

Contact & Admissions

Page avec les informations pratiques, le formulaire de contact, la FAQ et les informations d’accès au campus.

Fonctionnalités JavaScript ajoutées

Mode sombre

Un bouton permet de passer le site en mode sombre pour rendre la lecture plus confortable.

Carrousel

Sur la page d’accueil, plusieurs images défilent pour rendre la présentation plus vivante.

FAQ interactive

Les questions peuvent s’ouvrir et se fermer directement sur la page contact.

Formulaire de contact

Le formulaire vérifie que les champs importants sont bien remplis avant l’envoi.

Bouton retour en haut

Un bouton apparaît quand on descend dans la page pour revenir rapidement au début.

Effets visuels

Nous avons ajouté quelques animations et effets au survol pour rendre le site plus agréable.

Choix graphiques

Couleurs

Nous avons choisi des couleurs proches de l’univers EFREI, avec du bleu, du blanc et quelques touches de cyan et de jaune.

Style moderne

Nous avons voulu donner au site une apparence moderne, en lien avec l’informatique et le numérique.

Cartes

Les informations sont souvent présentées sous forme de cartes pour rendre les pages plus claires et plus faciles à lire.

Responsive design

Le site a été adapté pour pouvoir être consulté sur ordinateur, tablette et téléphone.

Difficultés rencontrées

Organiser les pages

Le plus difficile a été de bien répartir toutes les informations sans rendre le site trop chargé.

JavaScript

Certaines fonctionnalités ont demandé plusieurs tests, notamment le carrousel, le mode sombre et la FAQ.

Garder une cohérence visuelle

Nous avons aussi dû faire attention à garder les mêmes couleurs, les mêmes boutons et le même style sur toutes les pages.

Technologies utilisées

HTML

Pour créer la structure des pages et organiser le contenu.

CSS

Pour gérer le design, les couleurs, les cartes, les boutons et le responsive.

JavaScript

Pour ajouter de l’interactivité au site.

VS Code

Pour écrire et tester le code.
